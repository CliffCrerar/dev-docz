## Using Vscode extension:

#### 1. Installation is straight forward . . . .

#### 2. Where are the extensions installed. --> `C:\Users\<your windows username>\.vscode`

#### 3. To see your installed installations in vscode in the `search ext marketplace` type:

```
    @installed // shows all extensions currently installed

    @enabled // enabled extensions

    @disabled // disabled extnensions

    @installed `<type something>` // filters installed extension list
```
